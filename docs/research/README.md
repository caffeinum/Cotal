# Research

Background research that informs Cotal's design but isn't protocol spec. Unlike the rest
of `docs/` (which describes the wire contract), these are synthesis/position pieces.

- [readme-practices.md](readme-practices.md) — what 17 comparable repos (protocol specs,
  agent frameworks, dev-infra with hosted funnels) do in their READMEs, plus sourced
  best practices — distilled into recommendations for ours.
- [multi-agent-ux.md](multi-agent-ux.md) — how one human stays aware of and in control of
  many autonomous peers without being overwhelmed: human-AI interaction guidelines,
  cognitive-load / anti-overload, situational awareness, transparency & control, and how
  current agent tooling surfaces it — mapped onto Cotal's primitives with concrete
  recommendations.
