# User Guide

## Creating a task

Click **New task** above the task list. You'll see a form:

- **Title** — required, one short line.
- **Description** — optional, longer details.
- **Priority** — select **Low**, **Medium**, or **High**. Defaults to **Medium**.

Press **Add task** to create it. It appears immediately in the list.

## Marking a task complete

Click the checkbox next to a task to toggle completion. Completed tasks render with a
strikethrough but stay in the list.

## Deleting a task

Hover over a task and click the trash icon. Deletion is immediate.
