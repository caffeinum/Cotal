# cotal

A standard wire interface for software — especially AI agents — to coordinate in real
time as **lateral peers in a shared pub/sub space**, not as nodes in an orchestrator tree.

Name reserved. Implementation ships as the [`@cotal-ai/*`](https://www.npmjs.com/org/cotal-ai)
packages — see the [repository](https://github.com/Cotal-AI/Cotal).
